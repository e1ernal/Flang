# Flang

**macOSの入力ソース切り替えメニューに、国旗を取り戻す。**

[English](../../README.md) | [Español](README.es.md) | [Français](README.fr.md) | [Português](README.pt-BR.md) | [中文](README.zh-Hans.md) | 日本語 | [Русский](README.ru.md)

macOS 12.4までは、メニューバーのキーボードレイアウト切り替えメニューに国旗が
表示されていました。その後Appleは国旗を「ABC」「ES」のようなテキストラベルに
置き換えました。Flangは国旗を取り戻す小さなメニューバーアプリです——どの国旗
を、どの名前で、それぞれの言語に表示するかを自分で決められます。

## 機能

| | |
|---|---|
| メニューバーの国旗 | 現在の入力ソースを国旗付きで表示し、切り替えると即座に更新 |
| おなじみの切り替え方法 | クリックですべての入力ソースを表示し、システムメニューと同じように切り替え可能 |
| 柔軟なインジケーター | 国旗と名前を独立して表示——どちらか一方、両方、またはどちらも表示しない（その場合はシステムアイコンにフォールバック） |
| 2種類の国旗表示 | フラットな国旗画像（クラシックな見た目）、またはネイティブの絵文字国旗 |
| 妥当なデフォルト設定 | macOSのすべてのキーボードレイアウトと入力方式に、それぞれ妥当なデフォルトの国旗が設定される |
| 完全なパーソナライズ | 各言語について、任意の国旗・短い名前・正式名を自由にカスタマイズ可能 |
| 最小限のフットプリント | データ収集なし、ログイン時に自動起動、サイズは数メガバイト。唯一のネットワーク通信は任意の毎日の更新確認のみ |

言語は国に紐づいているわけではありません——だからこそFlangは国旗を個人の好み
で選べるようにしています。フランス語にカナダの国旗を、スペイン語にメキシコ
の国旗を使いたいですか？2クリックで完了です。

## インストール

[GitHub Releases](https://github.com/e1ernal/Flang/releases) から最新のzip
をダウンロードして展開し、Flang.appを「アプリケーション」フォルダに移動して
ください——開く前に、以下の内容を必ずご確認ください。

代わりにソースからビルドする場合：

```bash
git clone https://github.com/e1ernal/Flang.git
open Flang/Flang.xcodeproj
```

Cmd+Rでビルド・実行してください（Xcode 16以降、macOS 13 Ventura以降が必要です）。

### 未署名アプリを開く

FlangはApple公証（notarization）を受けていません。公証には有料のApple
Developer Programへの加入が必要ですが、このプロジェクトはまだ加入していません
——1.0で対価を払う価値のあるユーザーがいると証明できてから加入を検討する予定
です（[ロードマップ](#ロードマップ)参照）。それまでの間、macOSのGatekeeperは
通常のダブルクリックをブロックし、「"Flang"はmacOSによる悪意のあるソフトウェ
アのチェックを受けていないため開けません」と表示します。

これは想定内の動作であり、バグでも異常でもありません。初回起動時の手順：

1. 「アプリケーション」フォルダでFlang.appを右クリック（またはControlキーを
   押しながらクリック）し、**開く**を選択します。
2. 表示されるダイアログで再度**開く**をクリックします。

この操作が必要なのは最初の1回だけです——それ以降の起動（今後のアップデートを
含む）は通常どおり開きます。

## 使い方

1. Flangを起動すると、メニューバーに国旗が表示されます。
2. クリックすると入力ソースを切り替えられます。右クリックで「設定」と
   「終了」を選べます。
3. 任意：システム設定 — キーボードで「メニューバーに入力メニューを表示」の
   チェックを外すと、標準の切り替えメニューを非表示にできます。macOSはこれを
   アプリが自動で行うことを許可していないため、一度だけ手動で行う必要が
   あります。

   <img src="../images/hide-system-switcher.png" width="500" alt="システム設定 — キーボード。「メニューバーに入力メニューを表示」がハイライトされている">

4. キーボードレイアウトを追加・削除するには、「設定 — 入力ソース」を開いて
   「+」ボタン（または入力ソースの「削除」）を使用します——どちらも
   「システム設定 — キーボード — 入力ソース」に直接つながり、macOSが処理を
   行います。

   <img src="../images/add-input-source.png" width="500" alt="Flangの入力ソースタブ。「+」ボタンがハイライトされている">

## よくある質問

**Flangはシステムの切り替えメニューを置き換えますか？**
機能的には置き換えます。同じシステムAPIを使って同じ入力ソースを一覧表示・
切り替えします。システム自体のインジケーターは手動でしか非表示にできません
（「使い方」参照）。

**Flangにインターネット接続は必要ですか？**
いいえ。唯一の任意のネットワーク通信は、GitHub上の新しいバージョンを毎日
確認するものだけです。あなたやあなたのシステムに関する情報はどこにも送信
されません。
